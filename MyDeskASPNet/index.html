MyDesk ASP.NET Components

The ASP.NET components of Techlight MyDesk require compilation.

Setup Required:

The ASP.NET application needs to be built using Visual Studio.

Steps:
1. Open MyDeskASPNet.sln in Visual Studio
2. Restore NuGet packages
3. Build the solution (Ctrl+Shift+B)
4. Ensure the bin folder is created with compiled DLLs

Note: This ASP.NET site is separate from the main ASP Classic application.
You can use Techlight MyDesk ASP Classic without this component.

For development, the ASP Classic application at /Clients/SalesEngineTL/ contains the full functionality.
