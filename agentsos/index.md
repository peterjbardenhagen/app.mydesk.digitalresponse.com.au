# AgentsOS User Guide

AgentsOS is the AI orchestration layer inside MyDesk. Use it to submit project briefs, monitor AI agent runs, and interact with intelligent automations.

## Screens

### AgentsOS Main
![AgentsOS Main](screens/agentsos-main.html)

**Actions**
- Submit a new brief with title and description.
- Monitor running and completed projects.
- View real-time agent reachability status.

### AI Agents
![AI Agents](screens/ai-agents.html)

**Actions**
- Run individual agents (Lead Researcher, Quote Optimizer, Email Assistant).
- View run counts and average duration.
- Manage agent configurations and prompts.

### Ask AI
![Ask AI](screens/ask-ai.html)

**Actions**
- Chat with the AI assistant about business data.
- Ask questions about sales, quotes, invoices, and expenses.
- Use natural language to generate reports and insights.
